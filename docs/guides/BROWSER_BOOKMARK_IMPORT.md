# 浏览器书签 HTML 导入

在后台“导入 / 导出”页面选择或拖放浏览器导出的 `.html` / `.htm` 文件。文件只在浏览器中解析，原始 HTML 不会上传。

- 已知的“收藏夹栏”“Bookmarks Bar”“其他书签”等导出容器不作为业务分类；普通第一层文件夹会保留。
- 第一层有效文件夹映射为一级分类，第二层映射为二级分类；一级和二级分类都可以直接保存书签。
- 超过两层的剩余路径压平进二级标题，并使用 `›` 分隔。例如“开发 / 前端 / 框架”导入为一级“开发”和二级“前端 › 框架”。“前端”和“前端 › 框架”是同一父级下两个平级分类。
- 导出根目录或已忽略容器中的直属书签进入“浏览器书签”。
- 支持描述、添加时间、内嵌图片图标和 HTTP(S) 外部图标。
- 仅导入 HTTP(S) 链接，重复链接会保留。
- HTML 默认使用“追加合并”，也可明确选择覆盖。
- 合并时按去除首尾空格且忽略大小写的完整分类路径复用现有分类；不同父分类下的同名子分类彼此独立，站点设置保持不变。

普通网页无法直接读取浏览器收藏夹；请先从 Chrome、Edge 或 Firefox 导出标准书签 HTML。
