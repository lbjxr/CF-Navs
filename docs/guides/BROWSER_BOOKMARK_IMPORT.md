# 浏览器书签 HTML 导入

在后台“导入 / 导出”页面选择或拖放浏览器导出的 `.html` / `.htm` 文件。文件只在浏览器中解析，原始 HTML 不会上传。

- 顶层文件夹转换为分类，子文件夹中的书签并入对应顶层分类；根目录书签进入“浏览器书签”。
- 支持描述、添加时间、内嵌图片图标和 HTTP(S) 外部图标。
- 仅导入 HTTP(S) 链接，重复链接会保留。
- HTML 默认使用“追加合并”，也可明确选择覆盖。
- 合并时按去除首尾空格且忽略大小写的分类名复用现有分类，站点设置保持不变。

普通网页无法直接读取浏览器收藏夹；请先从 Chrome、Edge 或 Firefox 导出标准书签 HTML。
