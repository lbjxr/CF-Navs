# Archive

Holds local-only artifacts and reference snapshots not part of the CF-Navs application.

Contents:
- `sun-panel/` - local copy of the upstream Sun-Panel source used as the import-feature reference.
- `cf-navs-import.json` - local import data sample.
- `development-notes/` - local development plans, progress notes, and completed implementation reports.
- `browser-test-artifacts/` - temporary browser scripts and screenshots captured during UI verification.

This directory is git-ignored and is not part of the build or deployment.
